    ```
    curl -s -o /dev/null -w "%{http_code}" -X POST ${WORKER_URL}/ \
      -H "Content-Type: application/json" \
      -d '{"jsonrpc":"2.0","id":1,"method":"initialize","params":{}}'
    ```
    Atteso: `401`

    **Test 2 — OAuth token senza code → 400 (MCP-01b)**
    ```
    curl -s -o /dev/null -w "%{http_code}" -X POST ${WORKER_URL}/oauth/token \
      -d 'grant_type=authorization_code&code=invalid'
    ```
    Atteso: `400`

    **Test 3 — Auth valida → 200 initialize response (MCP-01c)**
    ```
    curl -s -X POST ${WORKER_URL}/ \
      -H "Content-Type: application/json" \
      -H "Authorization: Bearer ${MCP_BEARER_TOKEN}" \
      -d '{"jsonrpc":"2.0","id":1,"method":"initialize","params":{}}'
    ```
    Atteso: risposta JSON contenente `"protocolVersion":"2024-11-05"` e `"serverInfo"`

    Registrare i risultati nel SUMMARY (http status code per i test 1/2, excerpt JSON per test 3).
