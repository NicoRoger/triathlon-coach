# Injury Log

> Storico infortuni e fastidi muscolari/tendinei. Aiuta a riconoscere pattern
> ricorrenti (es. "sempre tendine d'achille dopo settimane >70km").

## Template entry

```
## YYYY-MM-DD — [zona corporea] — [stato: acuto|cronico|risolto]
- Trigger: cosa l'ha scatenato (sessione, contesto, calzature)
- Sintomi: dolore (0-10), localizzazione, quando
- Diagnosi medica: data, dottore, conclusione
- Trattamento: fisioterapia, esercizi, integrazione
- Stop allenamento: numero giorni / discipline
- Cross-training durante stop:
- Ripresa: data, gradualità, sensazioni primi giorni
- Lezioni:
  - Cosa lo ha causato (root cause)
  - Cosa fare diversamente
```

## Pattern noti
> Aggiornato dall'agente quando ricorrenze emergono dai dati.
